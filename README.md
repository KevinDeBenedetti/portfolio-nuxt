# Portfolio

Personal portfolio built with **Nuxt 4**, **TypeScript**, and **Tailwind CSS 4**.

## Tech Stack

- **Nuxt 4** — SSG, Vue 3.5+, TypeScript
- **Nuxt UI 4** — Components & dark mode
- **Nuxt Content 3** — Markdown with MDC syntax
- **i18n** — Bilingual (FR/EN)
- **Oxlint/Oxfmt** — Fast linting & formatting

## Quick Start

```bash
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm generate    # Build static site
pnpm check       # Lint + format + typecheck
pnpm preview     # Preview production build
```

## Project Structure

```
app/
├── components/   # Vue components
├── pages/        # File-based routing
└── composables/  # Composition functions

content/
├── en/           # English content
└── fr/           # French content
```
